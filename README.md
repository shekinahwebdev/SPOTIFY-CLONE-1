A music streaming application inspired by Spotify, built with React, Redux, and TypeScript. This project replicates core Spotify features like browsing songs, creating playlists, and playing music, while serving as a learning project for mastering frontend development.

### Features

- Play, Pause, Next, Previous controls
- Browse songs, albums, and artists

- Custom playlists (Create, Add, Remove tracks)

- Responsive design (Mobile & Desktop)

### Tech Stack

Frontend:

- React + TypeScript

- Redux (state management)

- React Router (navigation)

- Tailwind CSS
